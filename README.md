# Around The U.S. – React Version

## 📌 Description

This project is a React-based version of the "Around the U.S." application, originally developed using vanilla JavaScript.

The application allows users to interact with a dynamic gallery of images, including adding new cards, liking content, deleting items, and editing user profile information.

This version focuses on implementing a modern frontend architecture using React, improving code organization, scalability, and maintainability through reusable components and state management.

---

## 🚀 Technologies

- React
- Vite
- JavaScript (ES6+)
- CSS3
- Functional Components
- React Hooks (useState)

---

## 💡 Features

- Display a gallery of image cards
- Add new cards dynamically
- Edit user profile information
- Like and unlike cards
- Delete cards
- View images in a modal popup
- Responsive design (Desktop, Tablet, Mobile)

---

## 🔄 Improvements over Vanilla JS version

- Component-based architecture
- State management with React Hooks
- Cleaner and more maintainable code structure
- Reusable UI components
